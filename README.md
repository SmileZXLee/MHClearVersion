# MHClearVersion
## 注意：仅可用于学习交流娱乐之目的，不得用于商业用途。
## 安装
### 下载ipa文件，使用签名工具重签名，可使用可视化的[iOS App Signer](https://github.com/DanTheMan827/ios-app-signer)重签名。
### 使用Xcode或pp助手将ipa安装至手机即可，无需越狱。
